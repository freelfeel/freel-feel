Freel Feel Project Summary

Problem Description



With a growing student population the Freel Library needed to update its current manual tracking system for visitors to the library. The system operated using a physical clicker device whose numbers were then manually recorded. This system made referencing and organizing past collected data difficult.

System Capabilities



  • Log Visitors Entry
  • Store Accumulated Data
  • Provide Access to Data for Staff through organized Database Queries
  • Restart,Backup, and Export Daily CSVs to Library Dean

Library Benefits



With this new system in place the library will benefit by being able to understand patron entry on an annual,monthly,daily, or hourly level. This information can now be easily accessed for grants,funding, and institutional insight.
